# Personal Finance Tracker

A responsive dashboard to track **balance, savings, and daily expenses**.  
Built with **HTML, CSS, JavaScript, and Chart.js**, deployed via **GitHub Pages**.

## 📂 Structure
- `index.html` → Overview dashboard (root for GitHub Pages)
- `pages/saving.html` → Add account balances and savings
- `pages/payment.html` → Log daily expenses
- `css/style.css` → Shared styles
- `js/script.js` → Shared JavaScript logic
- `assets/` → Icons, images, favicon

## 🚀 Features
- Overview dashboard with balance, savings, statistics, recent activity, and last transaction
- Separate pages for adding savings and payments
- Dynamic Chart.js visualization of income vs expenses
- Responsive design for mobile browsers

## 📦 Deployment
1. Push repo to GitHub.
2. Enable GitHub Pages → Source: `main` branch → Folder: `/ (root)`.
3. Access at `https://<username>.github.io/finance-tracker`.
